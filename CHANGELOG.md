# Changelog

All notable changes to ImageEncryption project will be documented in this file.

## [1.0.0] - 2024-02-20

### Added
- Initial release of ImageEncryption
- AES-256 encryption/decryption for images
- Logistic chaotic map based pixel permutation
- PBKDF2 key derivation from passwords
- Command-line interface for encrypt/decrypt operations
- Support for multiple image formats (JPG, PNG, BMP, GIF, TIFF, WebP)
- Comprehensive unit tests
- Example usage scripts
- Full documentation and API reference

### Features
- Secure image encryption using AES-256
- Logistic chaotic map with configurable parameter
- Password-based key derivation
- Encrypted file format with metadata
- Proper padding and IV handling
- Cross-platform compatibility

- README with full usage guide
- API documentation
- Example scripts
